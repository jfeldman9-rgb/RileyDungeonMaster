# Riley vs Kenzie - Dungeon Master

Small Godot 4 fantasy adventure prototype starring Riley and Kenzie. The current pass is a compact open-valley/temple adventure with explorable routes, seal objectives, optional shrines, broccoli monsters, ninja-star combat, and a Kenzie boss encounter.

Open `project.godot` in Godot 4.6 or newer, then run the main scene.

Controls:
- Desktop: WASD/arrows move, Q/E turns the camera, Space slices, F throws ninja stars, Shift dashes.
- Mobile/web: left side is a virtual joystick, with Slice, Dash, and Star buttons on the right.

The world and gameplay are built procedurally in `scripts/game.gd`; generated sprites live in `assets/generated/`.
